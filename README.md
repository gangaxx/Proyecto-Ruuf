# Proyecto-Ruuf:
Proyecto desarrollado en python que simula una venta de helados por consola. El programa permite elegir la cantidad de bolitas por sabor y agregar salsas. También calcula el precio total de cada helado. Además, permite ingresar varios helados en un mismo pedido y muestra el monto final de los helados solicitados.


# Como funciona:
Al principio el programa da una bienvenida cordial, luego pide que el usuario ingrese un numero (este numero es el total de bolitas que desea en el helado) En las siguientes preguntas de las salsas este pregunta si desea agregar salsas o no al cliente (Aquí tendrá que escribir si o no desea agregar la salsa que pregunta el programa). Una vez finalizado... pregunta si desea agregar otro helado a la lista del pedido, si la respuesta es un no... el programa termina de preguntar cuando el cliente decida no agregar ningun otro helado a la lista del pedido y da el resultado final de la boleta y se despide de manera cordial.
